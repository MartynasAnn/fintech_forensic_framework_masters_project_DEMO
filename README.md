# Fintech forensic framework masters project DEMO

Python version required: 3.7+

## Installation instructions

1. Download the project's .zip file or clone through Git
2. Inside the project's folder create the virtual environment and activate it:
   ```
   python -m venv venv
   venv\Scripts\activate
   ```
3. Install dependencies:
   ```
   pip install -r requirements.txt
   ```
4. To launch GUI:
   ```
   streamlit run main.py
   ```
